# 🍷 Cave de Malta

Aplicativo simples para cadastro e organização de vinhos da adega pessoal.

## Funcionalidades

- Cadastro de vinhos
- Registro de vinícola
- Controle de safra
- Anotações de degustação
- Organização da coleção

## Próximas melhorias

- 📷 Anexar foto do rótulo
- ⭐ Avaliação do vinho
- 📍 Localização na adega
- 🔎 Pesquisa por nome ou vinícola
